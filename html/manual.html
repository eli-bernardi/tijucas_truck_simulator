Seja bem-vindo ao manual do Tijucas Truck Simulator! Este guia irá ajudá-lo a entender os controles, objetivos e dicas para aproveitar ao máximo sua experiência de jogo.

O objetivo do jogo é levar a carga até o fim do trajeto, coletando itens pelo caminho para aumentar sua pontuação.

Comandos

  • Seta direita / D — Mover para a direita
  • Seta esquerda / A — Mover para a esquerda
  • Seta cima / W — Desviar para cima
  • Seta baixo / S — Desviar para baixo
